```

	##> error

this didn't work because we had to run `ln` in the same directory as target `symlink`

opt02: run `ln` in same dir

``` 
mkdir c
cd c
ln -s ../a/f1 f1
readlink -f f1   
  ##> /Users/mertnuhoglu/projects/study/logbook/ex/log_20200517/e01/a/f1
