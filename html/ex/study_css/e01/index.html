Source: https://www.geeksforgeeks.org/css-cheat-sheet-a-basic-guide-to-css/?ref=rp#CSS%20Basics

1. Selektör: *

*(Universal) selektörü tüm arka fonu (background) pembe yapar 
      * {
          background-color: hsl(325, 63%, 82%);
          text-align: center;
      }

2. Selektör: span

span selektörü span elementini seçer
      span {
          background-color: skyblue;
      }
This span is styled using type selector.

3. Selektör: #div1

#div1 selektörü -> div id="div1"
      #div1 {
          color: green;
          text-align: center;
          font-size: 20px;
          font-weight: bold;
      }
This div is styled using id selector

4. Selektör: .div2

.div2 -> div class="div2"
    .div2 {
        color: orange;
        text-align: left;
        font-size: 25px;
        font-weight: bold;
    }
This div is styled using class selector

5. Selektör: div[style]

div[style] -> div style="color:green"
      CSS:
        div[style] {
            text-align: center;
            color: purple;
            font-size: 20px;
            font-weight: bold;
            margin-bottom: -20px;
        }

      HTML:

      < div style="color:green">
This div is styled using attribute selector

6. Selektör: < div>< p>

div>p -> div p
      div>p {
          color: #009900;
          font-size: 32px;
          font-weight: bold;
          margin: 0px;
          text-align: center;
      }
This div is styled using combinators

child selector

7. Selektör: .box:hover

.box:hover -> class="box" imleç üzerine gelince
      .box:hover {
          background-color: orange;
      }

pseudo selector:

My color changes if you hover over me!
